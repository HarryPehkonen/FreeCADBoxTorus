# FreeCADBoxTorus
Documenting the difficulty of creating a shape made up of a box and a torus in FreeCAD

## Procedure
Start the latest FreeCAD (1.1.1 as an AppImage).  New File --> Parametric Body.

Add an Additive Box.  Select the XY plane as the attachment.

Add an Additive Torus.  Select the YZ plane as the attachment.

The result doesn't look right.
